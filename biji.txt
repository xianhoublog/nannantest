git cancel operation
1. git commit --amend - 撤销上一次提交， 并将暂存区的文件重新提交
2. git  checkout -- filename 提取暂存区的文件， 并将其替换工作区的文件， 注意与git checkout branchname 的区别
3. git reset HEAD filename  拉取最近一次提交的版本库中的这个文件到暂存区， 该操作不影响工作区
